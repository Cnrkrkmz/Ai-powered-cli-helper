# stanctl — Air-gapped (offline/disconnected) environment

[air-gapped-package]
Creates the air-gapped package (run on a machine with internet access).
Usage: stanctl air-gapped package [flags]
Alias: stanctl agap
Version: --instana-release <number> OR --instana-version <version> (mutually exclusive)
Output directory: -o, --output-dir <dir>
License file: -l, --license-file <file>
Platform: -p, --platform linux/amd64 (default)
Disable cache: --no-cache
Registry access: --download-key or --registry-user/password/url

[air-gapped-import]
Imports the air-gapped package on the target system.
Usage: stanctl air-gapped import -f <package-file> [flags]
Required: -f, --file <archive-file>
Optional: --cluster-data-dir <dir>
