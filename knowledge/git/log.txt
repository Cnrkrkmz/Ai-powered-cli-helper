# git — Log and history

Short log: git log --oneline -20
Graph view: git log --oneline --graph --all
File history: git log --follow -p -- <file>
Filter by author: git log --author="Name"
Filter by date: git log --since="2024-01-01" --until="2024-12-31"
Search commit messages: git log --grep="search"
Search code changes: git log -S "search-string"
Specific file: git log --oneline -- <file>
