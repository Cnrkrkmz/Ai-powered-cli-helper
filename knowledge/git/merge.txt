# git — Merge and rebase operations

Merge: git merge <branch>
Squash merge: git merge --squash <branch>
No fast-forward merge: git merge --no-ff <branch>
Abort merge: git merge --abort
After resolving conflicts: git add <file> && git merge --continue

Rebase: git rebase <branch>
Interactive rebase (last 3 commits): git rebase -i HEAD~3
Abort rebase: git rebase --abort
Continue rebase: git rebase --continue
Skip current commit: git rebase --skip
