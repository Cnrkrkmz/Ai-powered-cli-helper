# stanctl — Debug, benchmark and dump

[debug]
Creates a debug package (logs, manifests, system info, healthchecks, metrics, config).
Usage: stanctl debug [-o <dir>] [-n <namespaces>]
Output: -o, --output-dir
Namespace filter: -n, --namespaces ns1,ns2
Disable compression: --no-compress

[benchmark-fio]
Runs FIO storage I/O benchmark. Tests all volumes if no directory is specified.
Usage: stanctl benchmark fio [flags]
Alias: stanctl bm
Single directory: --directory <path>
Default test directories: /mnt/instana/stanctl/{data,metrics,analytics,objects}

[dump-config-data]
Dumps config data for importing into another Standard Edition setup.
Usage: stanctl dump config-data [-o <dir>]
Alias: stanctl dm
Output: -o, --output-dir (default: current directory)
