# git — Branch operations

List all branches: git branch -a
Create branch: git branch <name>
Create and switch: git checkout -b <name>  or  git switch -c <name>
Switch branch: git checkout <name>  or  git switch <name>
Delete branch (merged): git branch -d <name>
Force delete branch: git branch -D <name>
Delete remote branch: git push origin --delete <name>
Rename branch: git branch -m <old> <new>
Rename current branch: git branch -m <new-name>
