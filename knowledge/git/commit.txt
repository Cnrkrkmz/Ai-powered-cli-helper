# git — Commit operations

Commit message must be in quotes: git commit -m "message"
Stage files first if nothing is staged: git add <file>
Empty commit: git commit --allow-empty -m "message"
Change last commit message: git commit --amend -m "new message"
Amend without changing message: git commit --amend --no-edit
Stage all tracked files and commit: git commit -am "message"
