# stanctl — Authentication management

[create-admin-api-token]
Creates an admin API token.
Usage: stanctl auth create-admin-api-token

[reset-2fa]
Resets two-factor authentication for a user.
Usage: stanctl auth reset-2fa -e <email>
Required: -e, --email <user-email>

[reset-idp]
Resets the IDP configuration. Switches back to Instana's internal user management.
Usage: stanctl auth reset-idp

[reset-password]
Resets a user's password. If not provided via flag, it will be prompted interactively.
Usage: stanctl auth reset-password -e <email> [-p <new-password>]
Required: -e, --email <email>
Optional: -p, --password <new-password>
