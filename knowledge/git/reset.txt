# git — Reset and undo operations

Soft reset (undo commit, keep changes staged): git reset --soft HEAD~1
Mixed reset (undo commit, changes become unstaged): git reset HEAD~1
Hard reset (CAUTION — changes are permanently deleted): git reset --hard HEAD~1
Hard reset to specific commit: git reset --hard <commit-hash>
Unstage a specific file: git reset HEAD <file>
Discard changes in a file: git checkout -- <file>
Create a new commit that reverts a commit (safe): git revert <commit-hash>
