# stanctl — Self-monitoring agent management

[agent-apply]
Installs or updates the Instana self-monitoring agent.
Usage: stanctl agent apply [flags]
Alias: stanctl ag a
Mode: --agent-mode INFRASTRUCTURE | APM | AWS (default: INFRASTRUCTURE)
Endpoint: --agent-endpoint-host, --agent-endpoint-port (443)
Key: --agent-key
Zone/Cluster name: --agent-zone-name (instana), --agent-cluster-name (instana)
Proxy: --agent-proxy-host, --agent-proxy-port (80), --agent-proxy-protocol (http|https)
       --agent-proxy-username, --agent-proxy-password, --agent-proxy-use-dns
Static image: --agent-use-static-image
Dry run: --dry-run

[agent-delete]
Deletes the self-monitoring agent.
Usage: stanctl agent delete
Alias: stanctl ag delete | del | rm | d
