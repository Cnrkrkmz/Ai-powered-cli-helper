# git — Push operations

Push new branch for the first time: git push -u origin <branch-name>
Push existing branch: git push origin <branch-name>
Force push (safe): git push --force-with-lease origin <branch>
Force push (destructive): git push --force origin <branch>
Push a specific tag: git push origin <tag-name>
Push all tags: git push origin --tags
Push all branches: git push --all origin
Delete remote branch: git push origin --delete <branch-name>
