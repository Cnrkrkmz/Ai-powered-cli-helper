# git — Stash operations

Save changes: git stash push -m "description"
Include untracked files: git stash push -u -m "description"
List stashes: git stash list
Apply (keep stash): git stash apply stash@{0}
Apply and remove: git stash pop
Delete specific stash: git stash drop stash@{0}
Delete all stashes: git stash clear
Open stash as branch: git stash branch <branch-name> stash@{0}
