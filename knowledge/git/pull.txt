# git — Pull ve fetch işlemleri

Pull (merge): git pull origin <branch>
Pull (rebase): git pull --rebase origin <branch>
Çakışma varsa: git rebase --continue veya git rebase --abort
Sadece fetch (birleştirme): git fetch origin
Tüm remote'ları fetch: git fetch --all
Silinmiş remote branch'leri temizle: git fetch --prune
