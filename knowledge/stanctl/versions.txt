# stanctl — Version and image management

[versions-identify]
Lists Instana versions compatible with the current installation.
Usage: stanctl versions identify [flags]
Alias: stanctl v id
Registry access required: --download-key or --registry-user/password/url

[versions-list-images]
Lists images for a specific Instana release.
Usage: stanctl versions list-images --instana-version <version> [flags]
Alias: stanctl v ls
Required: --instana-version <version>
